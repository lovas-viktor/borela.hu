# Vercel Deployment

**Live URL:** https://borela-hu.vercel.app/

## Auto-Deploy

The site automatically deploys from the `main` branch. Any push to GitHub triggers a production deployment.

- Push to `main` → Production deploy
- Push to other branches → Preview deployments
- Pull requests → Preview URLs generated automatically

## For Developers

Share the live URL with stakeholders, clients, or other team members. The site is always up-to-date with the latest `main` branch.
